# ANTIVIRUS - By  Noah Thiesset et Maxence Goutard

Le jeu Anti-Virus que vous voyez là est une version faite par Noah Thiesset et Maxence Goutard du jeu original de Smart Game.

Notre jeu se compose d'un écran d'accueil avec 4 niveaux de difficultés: Starter, Junior, Master et Expert *(Actuellement en cours de développement)*.

Une fois le niveau voulu choisie la partie se lance et vous devez faire sortir le virus à l'aide des flèches le faire sortir en haut à gauche.
Les touches utilisables sont :

Le pad numerique de 0 à 8,
* 0 étant le virus
* 1 étant le bleue 1
* 2 le orange,
* 3 le rose
* 4 le vert 1
* 5 le bleue 2
* 6 le violet
* 7 le vert
* 8 le jaune

Pour le déplacement, nous utilisons les flèches du clavier.

Pour la création du jeu, nous avons utilisé les assets du jeu original.
